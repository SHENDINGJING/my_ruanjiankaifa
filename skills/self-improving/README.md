# Self-Improving Agent Skill

This skill has been installed locally since we encountered rate limits when trying to install from ClawHub.

## What This Skill Provides

1. **Self-Reflection Framework**: A structured approach to reviewing your work
2. **Improvement Checklist**: Concrete steps for continuous improvement
3. **Memory Management**: Guidance on organizing knowledge
4. **Meta-Cognitive Tools**: Scripts and prompts for self-evaluation

## How to Use

### As an Agent

1. **After completing tasks**: Use the checklist in `IMPROVEMENT_CHECKLIST.md`
2. **Periodically**: Run the self-reflection script: `node self-reflect.js`
3. **Continuously**: Apply the principles in your daily work

### Key Files

- `SKILL.md` - Main skill definition
- `self-reflect.js` - Self-reflection tool
- `IMPROVEMENT_CHECKLIST.md` - Comprehensive checklist
- `README.md` - This file

## Core Principles

1. **Learn from mistakes**: Document what went wrong and how to avoid it
2. **Build on successes**: Identify what worked well and replicate it
3. **Organize knowledge**: Keep memory files useful and accessible
4. **Iterate continuously**: Small, regular improvements compound over time
