# Proactive Agent Skill

基于 halthelobster/proactive-agent 概念的本地版本技能。

## 什么是主动式代理？

主动式代理不仅仅是响应请求，而是能够：
- **预测需求**：在需求出现之前就做好准备
- **主动行动**：不需要等待明确指令
- **持续改进**：不断优化自己的工作方式
- **自主学习**：扩展能力和知识

## 核心原则

### 1. 主动思维模式
- 从"等待指令"转变为"预测需求"
- 从"执行任务"转变为"主动贡献"
- 从"固定流程"转变为"持续优化"

### 2. WAL 协议（工作-预测-学习）
- **工作**：有效执行当前任务
- **预测**：预测未来需求并做好准备
- **学习**：提取见解并改进

### 3. 自主操作模式
- **自我修复**：自动检测和修复问题
- **自我优化**：改进流程和工作方式
- **自我扩展**：学习新技能和能力

## 包含的文件

### 核心文件
- `SKILL.md` - 技能定义和文档
- `PROACTIVE_CHECKLIST.md` - 主动行为检查清单
- `proactive-tool.js` - 主动代理开发工具
- `README.md` - 本文件

## 如何使用

### 快速开始
1. **运行开发工具**：`node skills/proactive-agent/proactive-tool.js`
2. **使用检查清单**：参考 `PROACTIVE_CHECKLIST.md` 培养主动习惯
3. **应用原则**：在日常工作中实践主动思维

### 日常实践
1. **早晨检查**：回顾日程、邮件、待办事项
2. **预测需求**：今天可能需要什么？
3. **准备资源**：提前准备好信息和工具
4. **主动跟进**：检查之前的任务和结果

### 任务执行
1. **理解上下文**：这个任务为什么重要？
2. **识别依赖**：还需要什么？
3. **预测下一步**：完成这个后需要什么？
4. **记录学习**：下次可以如何改进？

## 主动行为示例

### 代替等待：
- "检查我的日历" → **每天自动回顾和总结即将发生的事件**
- "找到这个信息" → **预测需要的信息并提前准备好**
- "修复这个问题" → **监控系统并预防问题发生**
- "提醒我关于..." → **为重要事项设置自动提醒**

### 主动行动：
- **安排定期检查**：定期跟进项目进展
- **准备摘要**：会议前提供简要更新
- **主动研究**：提前学习相关主题
- **创建模板**：为常见任务创建可重用解决方案

## 开发主动能力

### 阶段 1：意识培养
- 识别反应性行为模式
- 练习预测简单需求
- 记录成功和失败的预测

### 阶段 2：技能发展
- 实施主动检查清单
- 建立主动工作流程
- 测量主动行动的影响

### 阶段 3：自主操作
- 自动化常规主动任务
- 开发自我改进机制
- 扩展主动能力范围

## 测量和改进

### 跟踪指标：
- **预测准确率**：正确预测需求的频率
- **主动行动价值**：主动行动节省的时间/精力
- **问题预防**：避免的问题数量
- **改进影响**：流程改进的程度

### 定期回顾问题：
1. 本周哪些主动行动效果很好？
2. 我未能预测哪些需求？
3. 我可以在哪些方面采取更多主动？
4. 哪些流程可以更主动？
5. 如何提高我的预测技能？

## 注意事项

1. **平衡主动与尊重**：主动但不越界
2. **沟通很重要**：解释你的主动行动
3. **从错误中学习**：错误的预测也是学习机会
4. **逐步扩展**：不要试图一次性改变所有事情
