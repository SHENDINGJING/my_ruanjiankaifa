# 🎯 股票分析交易与实时监控系统

基于OpenClaw AI助手开发的完整股票分析、交易策略和实时监控系统。

## 📋 项目概述

这是一个集成了股票分析、交易策略制定、实时监控和风险管理的完整系统。系统基于**中短线稳健型交易策略**，帮助投资者做出客观、纪律性的交易决策。

## 🚀 核心功能

### 1. 📈 股票分析系统
- **8维度基本面分析**：财务指标、行业地位、成长性等
- **技术分析**：趋势判断、量价关系、关键价位
- **投资组合管理**：仓位控制、风险分散
- **股息分析**：分红策略和收益计算

### 2. 🎯 交易策略系统
- **中短线稳健型策略**：明确的买入/持有/卖出规则
- **风险控制体系**：7%无条件止损 + 技术破位止损
- **盈利目标管理**：15%/20%分级止盈
- **标准化选股流程**：三步筛选法

### 3. 🔍 实时监控系统（3种方案）
- **方案1**：浏览器快照监控（OpenClaw集成）
- **方案2**：屏幕共享监控（节点设备）
- **方案3**：画布监控（专业界面，跨平台）

### 4. 📊 每日复盘系统
- 10分钟标准化复盘流程
- 大盘环境评估
- 板块强弱分析
- 自选股管理优化

## 🛠️ 技术架构

- **Python 3.8+**：核心分析引擎

### 系统特点
- ✅ **客观决策**：基于量化指标，避免情绪干扰
- ✅ **纪律执行**：明确的规则和流程
- ✅ **实时监控**：多方案确保及时响应

- ✅ **可扩展性**：模块化设计，易于扩展

## 🚀 快速开始

### 1. 环境准备
```bash
# 安装Python依赖
pip install yfinance pandas schedule

# 安装OpenClaw（如果尚未安装）
npm install -g openclaw
```

### 2. 运行股票分析
```bash
cd stock-analysis-project
python analyze_stock.py --symbol 601857
```

### 3. 启动实时监控
```bash
# 简单监控
python monitor_now.py

# 或使用OpenClaw集成监控
python openclaw_screen_monitor.py --monitor
```

### 4. 每日复盘
```bash
python daily_review.py --report
```

## 🎯 交易策略核心原则

### 1. 趋势优先原则
- 只做上升趋势，禁止抄底预测
- 20日均线作为趋势判断基准

### 2. 风险分散原则
- 单只股票 ≤ 20%仓位
- 单个行业 ≤ 30%仓位
- 总仓位动态调整

### 3. 风险控制原则
- 下跌7%无条件止损
- 技术破位立即止损
- 最大单次亏损 ≤ 7%

### 4. 能力圈原则
- 只做能理解的交易
- 放弃未知机会
- 专注优势领域
