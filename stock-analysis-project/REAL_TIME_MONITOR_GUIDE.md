# 股票实时监控系统 - 使用指南

## 🎯 系统概述

这是一个基于您的中短线稳健型交易策略的实时股票监控系统。系统可以：
- ✅ **定时监控**：每30分钟自动检查股票状态
- ✅ **智能提醒**：发现符合买入/卖出条件时立即提醒
- ✅ **风险预警**：接近止损价或触发止损时紧急通知
- ✅ **每日报告**：收盘后自动生成交易总结
- ✅ **OpenClaw集成**：通过您现有的消息渠道发送提醒

## 🚀 快速开始

### 1. 添加您的股票到监控列表

```bash
# 添加中国石油到监控
python stock_monitor.py --add "601857.SS,中国石油,800,12.465"

# 参数说明：
# 代码：601857.SS (A股代码加.SS后缀)
# 名称：中国石油 (可选，不填会自动获取)
# 股数：800 (您的持仓)
# 成本价：12.465 (您的成本)
```

### 2. 立即检查一次

```bash
# 手动检查所有监控股票
python stock_monitor.py --check

# 输出示例：
# 📈 中国石油 (601857.SS)
#    当前价: 12.160元
#    成本价: 12.465元 (亏2.45%)
#    止损价: 11.592元
#    目标价: 14.335元
#    💡 建议: 持有
```

### 3. 启动定时监控

```bash
# 启动每30分钟一次的监控
python stock_monitor.py --monitor --interval 30

# 监控期间会：
# 1. 每30分钟检查一次
# 2. 发现异常时打印提醒
# 3. 按Ctrl+C停止
```

## 📊 监控功能详解

### 监控检查内容
系统会检查每只股票的：

1. **价格变化**：当前价 vs 成本价
2. **止损触发**：是否跌破7%止损价
3. **目标达成**：是否达到15%/20%盈利目标
4. **回本状态**：是否回到成本价以上
5. **盈亏计算**：实时盈亏百分比

### 提醒级别
- 🚨 **紧急提醒**：止损触发、需要立即卖出
- ⚠️ **重要提醒**：达到盈利目标、回到成本价
- 📊 **状态提醒**：盈亏状态更新

### 监控时间安排
- **交易日**：09:30 - 15:00
- **检查频率**：每30分钟（可调整）
- **检查时点**：09:45, 10:15, 10:45, 11:15, 13:15, 13:45, 14:15, 14:45
- **每日报告**：15:30（收盘后）

### 2. 调整监控参数

编辑 `monitor_config.json`：

```json
{
  "monitoring_stocks": [
    {
      "symbol": "601857.SS",
      "strategy_settings": {
        "stop_loss_percent": 7,      // 止损比例
        "first_target_percent": 15,  // 第一目标
        "second_target_percent": 20  // 第二目标
      },
      "monitoring_frequency": "每30分钟"
    }
  ]
}
```

### 3. 设置定时任务（自动运行）

```bash
# 生成cron配置
python openclaw_integration.py --setup-cron

# 然后执行：
crontab stock-analysis-project/cron_setup.txt
```

需要配置OpenClaw的消息发送权限：

```python
# 示例代码（需要OpenClaw API支持）
def send_to_openclaw(message):
    # 调用OpenClaw消息API
    subprocess.run([
        "openclaw", "message", "send",
        "--channel", "feishu",
        "--message", message
```

## 🛠️ 故障排除

### 常见问题

#### Q1: 获取股票价格失败
**原因**：Yahoo Finance API限制或网络问题
**解决**：
```bash
# 检查网络连接
ping finance.yahoo.com

# 尝试其他数据源（未来扩展）
# 可以集成东方财富、新浪财经等国内数据源
```

#### Q2: 定时任务不运行
**检查**：
```bash
# 查看cron日志
tail -f /var/log/cron

# 检查cron服务状态
systemctl status cron

# 手动测试脚本
python stock_monitor.py --check
```

#### Q3: 编码问题（中文乱码）
**解决**：
```bash
# 设置UTF-8编码
export PYTHONIOENCODING=utf-8

# 或在Python脚本开头添加
# -*- coding: utf-8 -*-
```
